# Movil

Contendra la aplicacion cliente movil del proyecto NOS_VEMOS.

## Camara + IA

Se incluye una vista de prueba para usar la camara y enviar capturas al analizador IA por Pasarela:

- Archivo: `App_NosVemos_Movil/src/camara-ia.html`
- API objetivo: `http://localhost:7000/api/v1/ia/analizar-camara`
