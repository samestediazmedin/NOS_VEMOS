# Reportes Servicio

Genera consultas agregadas y exportacion de reportes operativos.

## Funcionalidades objetivo

- Consolidar metricas globales por servicio y por modulo.
- Construir paneles para seguimiento de casos, alertas y uso de IA.
- Exportar reportes a formatos tabulares (CSV/Excel/PDF en fases avanzadas).
- Consumir eventos para mantener modelos de lectura desacoplados.
