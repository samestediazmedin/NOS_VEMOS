## Cambios implementados

1. **Rama y entorno**

   - Se instalo `.NET SDK 9.0.313` para compatibilidad con `net9.0`.

2. **Auditoria Servicio**

   - Se agregaron endpoints:
     - `GET /health`
     - `GET /api/v1/auditoria/eventos`
     - `GET/POST /api/v1/auditoria/movimientos`
     - `GET/POST /api/v1/auditoria/asignaciones`
   - Se agregaron entidades de persistencia:
     - `AdminMovimiento`
     - `AsignacionPaciente`
   - Se agrego soporte `UseInMemoryDatabase=true` para ejecucion local sin SQL Server.
   - Se agrego dependencia `Swashbuckle.AspNetCore` para Swagger.

3. **Pasarela (Gateway)**

4. **Documentacion**

   - Se eliminó propuesta frontend de administrador por decision de producto.

5. **Frontend**

   - Se limpió referencia del modulo administrador en `App_NosVemos_Movil/README.md`.
