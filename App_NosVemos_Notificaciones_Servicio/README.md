# Notificaciones Servicio

Gestiona envio de correo, SMS o push a partir de eventos del sistema.

## Funcionalidades objetivo

- Plantillas de mensajes por tipo de evento.
- Reintentos y estado de entrega.
- Integracion asincrona via RabbitMQ desde NucleoNegocio y Orquestador IA.
- Historial de envios por usuario/caso.

## Estado

- Definido a nivel de arquitectura y documentacion.
- Implementacion API completa pendiente de fase de integracion.
